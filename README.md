# MONotepad

A notepad application bilt using C# and
WinForms.

## Requirements

- Allow the user to type and edit text.
- Support basic cursor movement and selection.
- Save the current text to a file. If the file is new, prompt the user for a filename and location (SaveFileDialog).
- Clear the current text area to start a new document.
- Load the contents of a text file (e.g., .txt) into the editor. This typically requires an OpenFileDialog control.
